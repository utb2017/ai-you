# Health

Primary home for health context: symptoms, routines, medications, doctors, therapy, sleep, mental health, and recurring body patterns.

## What goes here

- Doctors, therapists, specialists.
- Medications, dosages (general — redact exact prescription numbers if sensitive), and routines.
- Sleep patterns, anxiety patterns, recurring symptoms.
- Mental-health context that should carry between chats so the user doesn't have to re-explain.

## Privacy reminder

Even though this repo is private, the archive skill is told to redact:

- exact prescription identifiers,
- precise medical record numbers,
- anything that, if leaked, could harm the user.

The point is meaning, not raw medical data.
