# Inbox

Temporary holding area for notes that don't clearly fit any other category.

The archive skill is told to use this **only as a last resort**. If you see a lot of notes piling up here, that's a signal that:

- a new category is missing, or
- the archive skill needs a hint about where to file new notes.

You can move files out of `inbox/` into the right category at any time. The notes are plain Markdown.
