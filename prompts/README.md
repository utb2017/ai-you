# Prompts

These three prompts are the entire user surface of AI You. You paste them into ChatGPT.

| Prompt | When to use it |
| --- | --- |
| [`01-setup-ai-you.md`](01-setup-ai-you.md) | Once, after connecting GitHub and creating your private `ai-you-memory` repo. |
| [`02-archive-github.md`](02-archive-github.md) | At the end of an important conversation, to save the durable context. |
| [`03-recall-context.md`](03-recall-context.md) | At the start of a chat about a recurring person, project, or topic. |

After setup, you should rarely need to copy these again. The setup prompt asks ChatGPT to remember `Archive GitHub` and `Recall Context` as durable commands. From then on, you usually just type the command itself.

If a future ChatGPT session forgets the skill, paste the **long form** of the appropriate prompt and you'll be back in business.
