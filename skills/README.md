# AI You Skills

Skills are small command-style instructions that ChatGPT can read from GitHub.

## Core Skills

- [`archive-github`](archive-github/SKILL.md) - Save the current chat into the user's private memory repo.
- [`recall-context`](recall-context/SKILL.md) - Pull relevant context from the user's private memory repo before answering.

## Command Pair

- Use `Archive GitHub` to save.
- Use `Recall Context` to retrieve.
